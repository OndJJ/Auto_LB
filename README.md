# Auto_LB

Intflow와 진행한 Auto Labeling 프로젝트

오픈 소스로 제공된 Annotation_tools를 활용해 주석 생성 및 수정, 저장 기능을 프로젝트의 목표에 맞춰

MongoDB를 제거
서버로만 구동되게 수정 하는 작업 진행
